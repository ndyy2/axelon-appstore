# 🚀 Axelon Store

**Axelon Store** is a premium, high-performance Flatpak application manager designed specifically for Arch Linux. Built with **Electron**, **React**, and **Tailwind CSS**, it combines the power of the `flatpak` CLI with a stunning, modern GUI featuring glassmorphism aesthetics and real-time system feedback.

![Screenshot of Axelon Store](/public/icon.png)

## ✨ Features

- **🛍️ Intelligent Discovery**: Browse featured applications and categorized sections (Multimedia, Development, Games, and more).
- **🔍 Powerful Search**: Real-time search across thousands of Flatpak applications directly from Flathub.
- **📊 Real-time Metrics**: Detailed installation tracking with live download speed (MB/s), byte counters, and accurate ETA.
- **🔐 Permission Transparency**: Clearly understand app permissions (Network, Filesystem, Device access) before installing.
- **🖼️ Rich Previews**: High-resolution screenshot galleries and detailed application metadata.
- **🎨 Universal Theme System**: Seamless switching between **Light** and **Dark** modes with customizable accent colors (Violet, Blue, Emerald, Rose, Amber).
- **⚡ Performance**: Built on a reactive architecture for instant navigation and smooth animations.
- **🔄 Self-Update**: Built-in checker for Axelon Store updates via GitHub Releases.

## 🛠️ Technology Stack

- **Kernel**: Linux (Optimized for Arch Linux)
- **Runtime**: Electron (ESM)
- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS v4 (Semantic Variable System)
- **State Management**: React Context (Downloads & Themes)

## 📦 Installation

### Prerequisites

Ensure you have `flatpak` and `node` installed on your Arch Linux system:

```bash
sudo pacman -S flatpak nodejs npm
```

### Running for Development

1. Clone the repository:
   ```bash
   git clone https://github.com/axelon-automation/app-store.git
   cd app-store
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. In a separate terminal, start Electron:
   ```bash
   npm run start
   ```

### Building for Production

To create a production-ready package (AppImage or DEB):

```bash
npm run package
```
The output will be located in the `release/` directory.
